🎯 Comment configurer OBS (à faire une seule fois)
  1. Dans OBS → Sources → Ajouter → Navigateur (Browser)
  2. Coller l'URL du joueur dans le champ URL
  3. Largeur : 1920  Hauteur : 1080
  4. Cocher "Actualiser le navigateur quand la scène devient active"
  5. Activer le chroma key sur la source si tu veux un fond transparent
